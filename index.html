Baltimore City Crime Visulization

INTRODUCTION

In large cities, tracing crime patterns and analyzing them is a challenging spatiotemporal problem. This hardness is linked to two main factors:the sparse nature of crime activity and its widespread distribution.Generally hotspots will be classified by total number of crimes occurred in a particular location. But, areas with high probability can be more dangerous this is because of neglecting the possibility of crimes that is likely to occur. This problem can be solved by computing probability of a specific location. In this project we are going to specify a location is a hotspot by computing probability.

As graduate students(6030), we focused on relating problems to state-of-the-art research. We are interested in visualizing the crimes in Baltimore city. We picked a journal in IEEE named â€œCriPAV: Street-Level Crime Patterns Analysis and Visualization" as our reference.

OVERVIEW OF VISUALIZATION

By placing data in a visual context, such as maps or graphs, data visualization helps us understand what it means. This makes the data more natural to understand for the human mind, making it easier to see trends, patterns, and outliers in vast data sets.

DESCRIPTION OF OUR VISUALIZATION

We developed a scatter plot using d3 library where, we plotted a probability vs intensity scatter plot. We place a drop down to select a year from 2012 to 2017. We included 166 locations in each year. To develop a map we used leaflet which is an open source library of Java script and with the help of coordinates we got Baltimore city street level map and then created layers of map. Then we interacted a scatter plot with the map where, if we select a location in scatter plot it will only show that particular location where remaining locations will be disappear until we remove the cursor from that point. The user can also filter the data by choosing the locations in the map based on their required areas, and then the places will be filtered in the scatter plot.Location name, and the crime which occured most will be displayed on the map if we click a particular hotspot.
