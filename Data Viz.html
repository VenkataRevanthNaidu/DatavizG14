Visualization of crime data in Baltimore

Overview Of Visualization

By placing data in a visual context, such as maps or graphs, data visualization helps us understand what it means. This makes the data more natural to understand for the human mind, making it easier to see trends, patterns, and outliers in vast data sets.

Motivation to select this Field

This is the reason behind selecting this field. Data Set Explanation

The data set we use for crime visualization in Baltimore city, United States. Crime data set contains 2,76,350 crime incidents from 01/01/2012 to 09/02/2017. • Crime Date - Date of crime incident • Crime Time- Time stamp of crime incident • Crime Code- Crime code of each incident • Location- Location of Crime Incident • Description- Type of crime • Inside/Outside – Where the crime has occurred either inside or outside house • Weapon- Type of weapon for crime • Post- postal code of crime location • District- County of state • Neighborhood- Local landmark of crime incident • Longitude- Longitude of Crime • Latitude- Latitude of crime • Premise
